# <!Workout Community>
## サイト概要
### サイトテーマ
<筋トレ記録を共有するコミュニティサイト>
​
### テーマを選んだ理由
<私は筋トレを1年以上続けており、3分割法(胸・背中・脚)でトレーニングを行っています。
しかし、毎回のトレーニング内容(種目、重量、回数)を記録する習慣がなく、「前回より成長しているか」「どれだけ頑張ったか」が分からず困っていました。
また、同じように筋トレを続けている友人も「記録が続かない」「他の人の記録を見て参考にしたい」という課題を抱えていることを知りました。
初心者向けのアプリは多いですが、分割法を実践する中級者向けのシンプルな記録・共有サイトは少ないと感じています。
そこで、3分割法に対応した筋トレ記録を簡単に残せて、他のトレーニーとも共有できるコミュニティサイトがあれば、継続的なトレーニングの助けになると考え、このテーマにしました。

### ターゲットユーザ
- 筋トレ歴1年以上で3分割法を実践している人
- 自分のトレーニング記録を継続的に管理したい人
- 他のトレーニーの記録を見てモチベーションを上げたい人
​
### 主な利用シーン
- 筋トレ後にその日のトレーニング内容を記録したい時
- 過去のトレーニング記録を振り返りたい時
- 他のトレーニーの記録を見て参考にしたい時
